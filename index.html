ICS Date-Range Filter Bookmarklet

This bookmarklet lets you filter an .ics calendar file by date directly in your browser.

How to Install

  1. Drag the blue button below to your bookmarks bar, or right-click → “Bookmark link”.
  2. On any normal webpage, click the bookmarklet to run it.

How to Use

  1. Click the “ICS Date-Range Filter” bookmarklet in your bookmarks bar.
  2. Pick your Start and End dates.
  3. Click “Choose ICS & Filter” and select your .ics file.
  4. It downloads a filtered file to your Downloads folder.

Notes

  • The file never leaves your browser.
  • It works on any normal webpage but not on chrome:// or the new-tab page.
